SCP-8080 «СИЛУЭТ»

КЛАСС ОБЪЕКТА: KETER

Особые условия содержания: SCP-8080 находится в герметичном боксе с антигравитационной блокировкой. Прямой визуальный контакт запрещен без когнитивных фильтров.

Описание: SCP-8080 представляет собой гуманоидную тень, способную манипулировать когнитивным восприятием. При попытке наблюдения объект вызывает психологическую деградацию. Последние тесты показывают способность проецировать «фантомные воспоминания».

███████████ данные удалены. Для просмотра файлов инцидентов требуется уровень доступа 3+.

[ЗАПИСЬ СОВЕЩАНИЯ: объект шевелится даже при отсутствии света. Пересмотреть протоколы содержания.]

УСЛОВИЯ СОДЕРЖАНИЯ SCP-8080

Утверждено: Уровень 2/8080

  • Камера содержания: Зона-19C, стены из свинцово-полимерного сплава.
  • Обязательная смена когнитивных фильтров каждые 4 часа.
  • Датчики движения внутри камеры должны быть синхронизированы с AI-113.
  • Запрещено входить в одиночку. Минимальный уровень допуска для взаимодействия: 2.

Примечание: во время инцидента OMEGA протоколы были временно изменены.
